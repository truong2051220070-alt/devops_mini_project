# DevOps Mini Project - Frontend

Frontend xây dựng bằng React

## Tính năng

- ✅ Hiển thị danh sách tasks
- ✅ Tạo task mới
- ✅ Cập nhật trạng thái task
- ✅ Xóa task
- ✅ Xem thông tin sinh viên
- ✅ Responsive design
- ✅ Docker support

## Chạy locally

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Docker

```bash
docker build -t devops-mini-frontend .
docker run -p 3000:3000 devops-mini-frontend
```
